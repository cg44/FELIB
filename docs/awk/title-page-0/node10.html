- Assembles the imaginary part of a symmetric complex system matrix

- Assembles the imaginary part of a unsymmetric complex system matrix

- Assembles a full system matrix

- General assembly of a full system matrix

- Assembles the right-hand side of a system
ASSYM

ASSYMG

ASUSM
- Assembles a unsymmetric system matrix
ASUSMG
- General assembly of an unsymmetric system matrix

- Assembles the real part of a complex right-hand side
RASSYM
- Assembles the real part of a symmetric complex system matrix
RASUSM
- Assembles the real part of a unsymmetric complex system matrix
