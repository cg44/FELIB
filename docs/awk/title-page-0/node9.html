Stress-Strain Matrices

DAXI
- Forms stress-strain matrix for axisymmetric elasticity
DISO
- Forms stress-strain matrix for 3-D elasticity
DPLT
- Forms stress-strain matrix for 2-D plate bending
DPSN
- Forms stress-strain matrix for plane strain
DPSS
- Forms stress-strain matrix for plane stress
