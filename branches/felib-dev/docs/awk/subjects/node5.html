Numerical Integration

BQBRK
- Forms quadrature rule for line integration on hexahedrals
BQTRI
- Forms quadrature rule for line integration on triangles
BQQUA
- Forms quadrature rule for line integration on quadrilaterals
LINTRI
- Calculates a unit length on the boundaries of triangles
LINQUA
- Calculates a unit length on the boundaries of quadrilaterals
QBRK6
- Six-point quadrature rule for cube
QBRK8
- Eight-point quadrature rule for cube
QLIN2
- Two-point, one-dimensional quadrature
QLIN3
- Three-point, one-dimensional quadrature
QQUA4
- Four-point quadrature rule for rectangular region
QQUA9
- Nine-point quadrature rule for rectangular region
QTET4
- Four-point quadrature rule for tetrahedral region
QTRI4
- Four-point quadrature rule for triangular region
QTRI7
- Seven-point quadrature rule for triangular region
QWDG6
- Six-point quadrature rule for wedge-shaped region
QWDG8
- Eight-point quadrature rule for wedge-shaped region
SURBRK
- Calculates a unit of area on boundary faces
